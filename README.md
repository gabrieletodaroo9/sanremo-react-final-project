# Sanremissimo - Frontend

Il frontend di Sanremissimo è una Single Page Application (SPA) reattiva che permette agli utenti di esplorare la storia del Festival di Sanremo. L'interfaccia interroga le API del backend Laravel per visualizzare le edizioni passate, i dettagli dei cantanti, le canzoni in gara e i premi assegnati, offrendo un'esperienza di navigazione fluida e dinamica.
